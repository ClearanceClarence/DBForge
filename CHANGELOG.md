# Changelog

All notable changes to DBForge are documented in this file.

---

## [1.2.0] — 2026-04-14

### Added
- **First-run installer** — 3-step setup wizard (database connection → admin account → done). No default credentials — users must set a password on first launch
- **Settings page** — edit all config options from the UI: database connection, theme, rows per page, auth toggles, user management, IP whitelist, hidden databases, and more. Writes directly to `config.php`
- **User management** — add/remove users, change passwords (bcrypt hashed), enforce minimum password length from the Settings page
- **Bulk row selection** — checkbox column with select-all (supports indeterminate state), bulk action bar with "Delete Selected" and "Clear". Sends a single `DELETE ... WHERE IN (?)` query
- **Editable table structure** — click the pencil icon on any column to edit name, type (searchable datalist with common MySQL types), nullable, default, extra, and comment. Save runs `ALTER TABLE ... CHANGE`
- **Add/drop columns** — add new columns with position control (FIRST, AFTER x, or at end). Drop columns with confirmation modal
- **AUTO_INCREMENT display** — shows current next-ID value on the Structure tab with Set (custom value) and Reset (MAX+1) buttons
- **Foreign key visualization** — queries `information_schema` for FK relationships. Structure tab shows PK/FK indicators with gold/blue left borders, icon badges, and clickable FK links. Dedicated Foreign Keys and Referenced By tables with ON DELETE/ON UPDATE color-coded badges (CASCADE=red, RESTRICT=gray, SET NULL=amber)
- **Column type toggle** — button in browse toolbar to hide/show column types under headers, persisted to cookie
- **Auto-sizing inline editor** — cells with large text (>60 chars or multiline) open a resizable textarea instead of a single-line input. Auto-grows to fit content (max 300px). Ctrl+Enter saves in textarea mode, Enter saves in single-line mode
- **Syntax-highlighted CREATE statement** — the Structure tab's CREATE TABLE output is tokenized and highlighted using the same SQL engine as the editor
- **Redesigned Export page** — works with or without a table selected. Shows SQL and CSV export cards with row counts, size estimates, and filename on the download button. Includes a per-table export list for the entire database
- **Redesigned Info page** — header card with table name and key stats, three grouped sections (Properties, Storage, Timestamps) using key-value tables, prominent Danger Zone with descriptions
- **Redesigned Server page** — same grouped section pattern: MySQL Server, Performance, PHP Environment. Header card with hostname and stat counters
- **Server overview on home page** — when no database is selected, shows a phpMyAdmin-style dashboard with all databases listed in a table (table count, rows, data/index/total size, collation, per-row actions)
- **30+ inline SVG icons** — Lucide-style icon library via PHP `icon()` helper. Used across all templates: header, tabs, sidebar, toolbar, buttons, badges, modals. No CDN, works offline
- **Custom modal dialogs** — `DBForge.confirm()` and `DBForge.alert()` replace all native `confirm()`/`alert()` calls. Themed, animated, keyboard-navigable (Enter/Escape), backdrop blur

### Security
- **11-step security chain** — security headers → HTTPS enforcement → IP whitelist → session hardening → authentication → brute force protection → CSRF tokens → read-only mode → hidden databases → query logging → .htaccess rules
- **Authentication system** — themed login page, multi-user support, bcrypt password hashing, session timeout with periodic ID regeneration
- **Brute force protection** — IP-based lockout after configurable failed attempts
- **CSRF protection** — tokens on all POST forms and AJAX requests, validated server-side
- **Read-only mode** — blocks all write queries (INSERT, UPDATE, DELETE, DROP, ALTER) at both UI and API level
- **Query audit logging** — logs every query with timestamp, username, database, IP, and execution time
- **`.htaccess` hardening** — blocks direct access to config.php, includes/, logs/; disables directory listing; sets security headers

### Changed
- Config file is now generated by the installer, not shipped with defaults
- `config.template.php` is the base template used by the installer
- Auth is always enabled for new installs (no more `require_auth: false` default)

## [1.0.0] — 2026-04-14

### Initial Release
- Browse tab with paginated data grid, search, sort by column, type-aware cell styling
- Inline cell editing with AJAX saves, NULL support, Enter/Tab/Escape keyboard shortcuts
- Structure tab with column definitions and index viewer
- SQL editor with real-time syntax highlighting (custom tokenizer, 12 token types)
- Context-aware autocomplete (tables after FROM, scoped columns after WHERE, dot-notation alias resolution)
- Two-pass table/alias recognition in SQL queries
- Info tab with table metadata
- Export tab with SQL and CSV download
- Server info page with MySQL and PHP environment details
- 10 built-in themes (5 light, 5 dark) with cookie persistence
- Theme system with auto-discovery from `themes/` directory
- Database sidebar with exact `COUNT(*)` row counts
- Quick query links on the Info tab
- Status bar with connection info and keyboard shortcut hints
- Pure PHP + vanilla JS, zero external dependencies
- Drop-in install on XAMPP/WAMP/MAMP
