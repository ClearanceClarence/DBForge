Features
Everything you actually need.
Not a stripped-down toy. Not a bloated framework. Every feature is something a developer actually uses, built without a single npm install.
✏️
Real SQL Editor
Custom 612-token tokenizer with context-aware autocomplete, alias tracking, line numbers, and persistent drafts. Not a textarea with color — a real editor.
🔗
FK Drill-Down
Click any foreign key value to jump to the referenced row. Automatic linking, exact-match filters, cross-database support. Navigate relationships without writing SQL.
⚡
EXPLAIN Visualizer
One-click query plan analysis. Color-coded access types, warning badges on filesort and temporary tables, row count highlighting. Understand why queries are slow.
📊
ER Diagram
Interactive SVG with crow's foot notation, three line styles, force-directed layout, auto-save per database. Right-click context menus. Drag, zoom, hover to trace relationships.
👁
Views, Triggers, Routines & Events
Full CRUD for every MySQL schema object. Syntax-highlighted editors with skeleton templates. Scheduled events with live scheduler-status detection and one-click enable/disable.
🔍
Search Across Tables
Find a value in every table in the database simultaneously. Results grouped by table with matching columns highlighted and "View all" links.
📝
Inline Editing
Click any cell, type, press Enter. AJAX save — the page never reloads. Auto-switching textarea for long content. Bulk select + delete with confirmation.
🔧
Operations & Maintenance
Alter engine, collation, row format. Rename, move, copy tables. Optimize, analyze, check, repair in one panel. Partition management: create, drop, truncate, rebuild.
⚡
Live Processes
SHOW FULL PROCESSLIST with 2/5/10s auto-refresh. Color-scaled query duration (gold at 3s, red at 60s). One-click kill with self-session guard. Filter, hide-sleeping toggle, current connection marker.
⭐
Saved Queries & Favorites
Save queries with names for instant reuse. Star tables for quick sidebar access. Persistent SQL drafts. Row detail panel for wide tables. Everything a daily driver needs.
🐳
Docker Ready
docker-compose up and you're running. DBForge + MySQL 8.0 with healthchecks, persistent volumes, and a quick-start guide. From zero to managing databases in 30 seconds.

Security
12 layers. Not bolted on.
From bcrypt hashing to TOTP 2FA, every layer is built in. No default passwords. No "restrict at the web server level" nonsense.
🔐
TOTP Two-Factor Auth
Optional per-user TOTP. Scan a QR code with Google Authenticator, Authy, or 1Password. Verify with a 6-digit code on every login.
🔑
Bcrypt Authentication
Multi-user support with bcrypt password hashing. First-run installer — no default admin/admin ever exists.
🛡️
CSRF Protection
Token generated per session, validated on every POST form and AJAX request. No exceptions.
🚫
Brute Force Lockout
IP-based lockout after configurable failed attempts. Both password and 2FA failures count.
🔒
Read-Only Mode
Regex detection of write keywords. Blocked at UI and API level. Safe for junior developers.
📋
Audit Logging
Every query logged with timestamp, username, database, IP, and execution time.
Get Started
Three steps. Under a minute.
1
Clone or download
Drop the dbforge/ folder into your web root.
2
Open in browser
Navigate to localhost/dbforge/ — the installer launches automatically.
3
Configure
Enter MySQL/MariaDB credentials, create an admin account. That's it. You're in.
# Option 1: Clone
$ git clone https://github.com/ClearanceClarence/DBForge.git
$ cp -r DBForge/dbforge/ /var/www/html/
$ open http://localhost/dbforge/
# Option 2: Docker (recommended)
$ git clone https://github.com/ClearanceClarence/DBForge.git
$ cd DBForge && docker-compose up -d
$ open http://localhost:8080/dbforge/
